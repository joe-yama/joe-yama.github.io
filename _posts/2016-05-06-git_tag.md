gitのタグ機能を使ったので自分用に使い方をメモしておきます．すごくかんたん

## git tag？

一言で言えばコミットに目印をつけること

詳しくはは[このページ](https://git-scm.com/book/ja/v1/Git-%E3%81%AE%E5%9F%BA%E6%9C%AC-%E3%82%BF%E3%82%B0)がすごく分かりやすかったので参考にしてください．

## 使い方

### 現在あるタグを確認

```
git tag
```


### 軽量版タグ

```
git tag [tag name]
```

これだけ．

### コメントつきタグ

```
git tag -a [tag name]
```

エディタが開くのでそこにコメントを記述．

### ワンライナーのコメント付きタグ

```
git tag -am "comment!" [tag name]
```
コミットと同じ要領ですね．

## リモートに反映

**ふつうにpushするだけではタグはリモートリポジトリに反映されない．**

```
git push origin [tag name]
```

これでタグがpushされる．
